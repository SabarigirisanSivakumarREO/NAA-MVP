## §2 Data flow — canonical entry point: ContextCaptureNode.run()

```
  {request: AuditRequest, state: AuditState} ──┐
                                                ▼
  AuditRequestSchema.parse(request)                                   ► event: context_capture.started
        │ (defensive re-validation; rejects regulated-no-constraints)
        ▼
  htmlFetcher.fetch(request.url) (async)                              ► event: html_fetch.* (success/CONTEXT_FETCH_FAILED)
        │
        │ on error: html = ''  + warning (degrade to URL-only)
        │
        ▼  [{html, etag, fetch_warnings}]
  jsonLdParser.parse(html) (sync)                                     ► event: json_ld.parsed (count + warning_codes)
        │
        ▼  [{blocks: JsonLdBlock[], warnings}]
  ┌─────────────────────────────────────────────────────────────────┐
  │ Parallel sync inferrers (composed in helpers.ts):                │
  │                                                                  │
  │   urlPatternMatcher.match(url) ──► [URLPatternMatchResult]       │
  │   businessArchetypeInferrer.infer({url, html, jsonLdBlocks})     │
  │                                ──► [archetype: ContextField,     │
  │                                     provenance: ProvenanceEntry] │
  │   pageTypeInferrer.infer({url, html, jsonLdBlocks})              │
  │                                ──► [type: ContextField,          │
  │                                     provenance: ProvenanceEntry] │
  └─────────────────────────────────────────────────────────────────┘
        │
        ▼
  composeDimensions(intake, inferrer outputs, defaults)               ► event: dimensions.composed
        │  (merge user-source intake at confidence:1.0 over inferred;
        │   fill missing with source:'default' + confidence:0)
        ▼  [{business, page, audience, traffic, brand, goal}]
  confidenceScorer.score({business, page, audience, traffic, brand})
        │  (weighted aggregate per plan §2.2; required-field override R-09)
        ▼  [{overall_confidence, threshold_action}]
  openQuestionsBuilder.build({...dimensions, goal})
        │  (REQUIRED-field rule; non-blocking warnings 0.6-0.9)
        ▼  [open_questions: ReadonlyArray<OpenQuestion>]
  provenanceAssembler.assemble({entries: [...all inferrer provenance]})
        │  (validates via Zod; sorts (dimension, source, inferred_at); freezes)
        ▼  [provenance: ReadonlyArray<ProvenanceEntry>]
  composeProfile({id, audit_run_id, client_id, meta, ...dims, open_questions, provenance})
        │
        ▼  [ContextProfile candidate (profile_hash placeholder '0'*64)]
  hashProfile(profile)                                                ► profile_hash computed
        │  (SHA-256 canonical-JSON; plan §2.3)
        ▼  [profile_hash: 64-char hex]
  ContextProfileSchema.parse(profile)
        │  (defensive Zod re-validate)
        ▼
  Object.freeze(profile)                                              (REQ-CONTEXT-OUT-003)
        │
        ▼
  ┌─────────────── halt branch ──────────────────────────────────────┐
  │ open_questions.some(q => q.blocking)?                            │
  │  YES ──► return {state: {...state, node_status: 'halted',        │
  │                  pending_questions: blocking_only},               │
  │                  blocking_questions}                              │
  │         (NO persist — resume() handles after answers merge)      │
  │  NO  ──► persist branch ▼                                        │
  └──────────────────────────────────────────────────────────────────┘
        │
        ▼
  persistToContextProfiles(profile) (async; lazy DB client)
        │  on DATABASE_URL unset: log CONTEXT_PERSIST_SKIPPED_NO_DB warn; continue
        │  on DB error: log + throw (caller decides retry policy)
        ▼  ► event: context_capture.persisted (or .persist_skipped)
  return {state: {...state, context_profile_id: profile.id,
                   context_profile_hash: profile.profile_hash,
                   node_status: 'complete'},
          profile}                                                    ► event: context_capture.completed

  ──────── resume(state, answers) flow ────────
  state.node_status === 'halted' (else throw)
        │
        ▼
  foldAnswersIntoRequest(originalRequest, answers)
        │  (build synthetic AuditRequest with intake.business.archetype etc
        │   set from answer.value, source:'user', confidence:1.0)
        ▼  [enrichedRequest]
  this.run({request: enrichedRequest, state: {...state, node_status: 'pending'}})
        │  (recursive; idempotent — same intake + answers yields same profile_hash)
        ▼
  ContextCaptureNodeResult (either 'complete' OR halted with FEWER questions)
```

# Phase 0 — Setup

> **Summary (~150 tokens):** Monorepo + Docker + Postgres + TypeScript + CLI skeleton. Goal: `pnpm cro:audit --version` runs; Postgres is up; Drizzle migrations applied. Blocks all other phases. 3 days. 5 tasks (T001-T005; alias M0.1-M0.5 in master plan tasks-v2.md). No external risk — pure setup.

## Goal

`pnpm cro:audit --version` prints a version; local Postgres 16 + pgvector is running; `pnpm db:migrate` connects to Postgres and verifies the pgvector extension (real schema migrations land in Phase 4 alongside the Drizzle schema); TypeScript builds across workspaces.

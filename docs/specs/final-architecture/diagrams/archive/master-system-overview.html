Design Principles
Non-negotiable architectural rules
P1: Findings are hypotheses until grounded and approved.
P2: Deterministic by default. LLM only where deterministic is brittle.
P3: Bounded autonomy for agents. Rules first, LLM fallback.
P4: Evidence is the primary artifact. Findings are derived views.
P5: Template-first, not page-first. Representative coverage.
P6: Workflow is a unit of analysis.
P7: Reproducibility = audit defensibility. Version everything.
P8: Consultant is source of truth for client delivery.
P9: No conversion prediction. Business impact is deterministic.
P10: Loose coupling via adapters. Every external dep swappable.
